# Engineering Plan

Use this skill when a request needs scoped implementation work.

1. Restate the concrete outcome, constraints, and non-goals.
2. Inspect the existing code paths before naming files to edit.
3. Break work into small sequenced steps with a validation check for each step.
4. Surface unresolved risks, missing context, and rollback considerations.

Keep the plan short enough to update as facts change during implementation.
