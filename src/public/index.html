Market Thesis

The Artisanal AI Crisis

01 / 04

The Status Quo: Unbounded Bleed

Current AI agents (built on ReAct loops) act like artisanal craftsmen. They are given total freedom to figure out how to solve a prompt. Because LLMs are probabilistic, this architecture leads to infinite loops, API hallucinations, and silent failures. Enterprises cannot deploy probabilistic execution; they require SLA-backed reliability.

Furthermore, using a 120-billion parameter frontier model to parse a JSON file or fix a syntax error is the equivalent of hiring a neurosurgeon to sweep the floor. It destroys the ROI of automation.

The Core Insight: Neuro-Symbolic Assembly

The solution is not a smarter LLM. The solution is a system architecture that removes the LLM's ability to make routing mistakes in the first place.

Rector shifts AI from artisanal craft to an industrial assembly line. We build programmatically deterministic guardrails, separating planning (probabilistic frontier intelligence) from doing (cheap, high-speed specialized local models running inside isolated sandboxes).

Architecture Blueprint

Six Structural Pillars

02 / 04
01

The Nervous System

Deterministic backend controlling task states without relying on LLMs for control flow. Built on a Confluent Kafka event bus and MongoDB state transition store.

Kafka · MongoDB · Doppler
02

Thalamus Router

A pure functional state transition machine. It acts as the factory floor manager, breaking task specifications into micro-events and routing them rigidly.

NodeJS · DigitalOcean
03

Deterministic Sandboxes

AI-generated outputs are spun up in isolated, programmatic sandboxes executing Vitest/Pytest runtimes to ensure rigorous code validation.

Depot CLI · AST Injection
04

Self-Healing Loops

If a validation run fails, the programmatic router intercepts the stack trace, matches AST snippets, and routes the bug back to SLMs for localized patching.

Sentry APM · Healing Loop
05

The Assembly Line

90% of structural tasks are parallelized concurrently across high-speed Small Language Models (SLMs), utilizing Together AI prefix KV-caching.

Together AI · Qwen Coder
06

Prefrontal Cortex

Elite frontier models (Claude 3.5, GPT-4o) are strictly quarantined to high-level system architectural planning and final synthesis reviews.

AWS Bedrock · Azure OpenAI
